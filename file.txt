Grunt Install
En la terminal
sudo npm install -g grunt-cli
En el proyecto
npm i grunt --save-dev
Se crea el archivo en el proyecto
touch Gruntfile.js
Instalacion de libreria para comprimir
npm i grunt-contrib-uglify --save-dev


Instalacion de Angular 2
npm i -g angular-cli
Creacion de proyecto
ng new primer-proyecto
Lanzar el primer proyecto
ng serve
url de angular
http://localhost:4200

para crear un componente
ng g component barra-superior
